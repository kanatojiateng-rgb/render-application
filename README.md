## Development Background
近年、投資や資産形成が注目されています。
しかし、多額の資金を動かせるのは一部の資産家に限られ、
一般的な世帯や若年層にとっては「日々の生活で手一杯」という状況も少なくありません。
特に、ローン返済や無意識な支出による貯蓄不足など、
いわゆる「浪費の構造」は現代において深刻な課題だと感じています。
本アプリは、ITの力を活用し、
日々の小さな支出を見直すことで生まれる節約額が、
将来的にどれほどの資産価値へと変化するのかを可視化するこ���を目的としています。
大きな資金がなくても、
「節約 → 積立 → 複利運用」というプロセスを通じて
将来資産を形成できることを視覚的に示し、
生活習慣を見直すきっかけを提供するWebアプリです。

## Features
- 支出の登録
- 月次・年間支出傾向の把握
- 節約額の算出
- 複利計算による将来資産形成シミュレーション

## Investment Simulation
フロントエンドで入力されたデータをFlask経由でバックエンドに渡し、SQLAlchemyでデータ管理を行っています。また、年間節約額を一定利回りで複利運用した場合の将来資産額をシミュレーションします。


## Tech stack
Frontend： HTML/css/JavaScript
Backend : Flask(Python)
Database : SQlite
ORM : SQLAlchemy
